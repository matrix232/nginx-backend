## Запуск

Выполнить заполнение файла окружения, скопировав его и вставив порт:
```bash
cp .env.example .env
```

```bash
docker compose up -d --build
```

---

## Проверка

После запуска выполнить команду или выполнить открытие в браузере http://localhost:

```bash
curl http://localhost
```

Ожидаемый результат:

```
Hello from Effective Mobile!
```

---

Происходит обращение к веб-серверу, nginx принимает запрос по 80 порту и далее проксирует запрос на бекенд, обращаясь по имени контейнера в upstream. Бекенд сервис обрабатывает запрос и возвращает строку.
